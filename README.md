# A3MC Kerberos

## Version 0.1

[A3MC Homepage](https://a3m-community.de)

[Steam Seite](http://steamcommunity.com/groups/A3M-Community)

[Discord](https://discordapp.com/invite/mEfXAEd)

Hier findet ihr die Dateien zu unserer Kerberos Version, angepasst auf die A3M-Community.

Original Beschreibung:

Operation Kerberos is a dynamic Arma 3 COOP-mission.   
The players have to succeed during serveral Missions. The goal is to play together as a team against an virtual enemy commander.   
To achive this, the human player can use serveral supporting features.
